# MeetMind AI Frontend

MeetMind AI is a React + Vite interface for recording meetings, uploading audio, diarization review, and AI-generated summaries.

## Development

- Install: `npm install`
- Run: `npm run dev`
- Build: `npm run build`

## API

In development, requests to `/api` are proxied to `http://127.0.0.1:5000`.
You can override the API base with `VITE_API_BASE_URL`.
